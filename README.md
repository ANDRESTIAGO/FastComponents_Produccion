# FastComponents_Produccion

En este proyecto se utilizó el lenguaje Python para el desarrollo general.
Se empleó FastAPI como framework para la implementación de endpoints y Jinja para la integración entre el backend y el frontend.

La aplicación fue desplegada mediante el servicio Render, mientras que Clever Cloud se utilizó para el alojamiento de la base de datos. Además, se utilizó GitHub para el control de versiones.

Como apoyo al proceso de desarrollo, se aplicaron buenas prácticas que incluyeron el uso de herramientas como Jira y Miro para la gestión y organización de ideas.

Asi mismo, se implementaron metodologías ágiles como Scrum y Kanban para asegurar la entrega oportuna y progresiva de los sprints.

En estos tiempos, muchas personas desean armar su computadora de escritorio, el problema es que compran los componentes sin saber si son compatibles o no entre si, haciendo que el propio usuario pierda dinero y tiempo, por esa razon la pagina brinda la solicion de un filtro en donde los componentes incompatibles con la placa, son quitados de las opciones a elegir.

El objetivo de la pagina es brindar una forma mas sencilla y sin equivocaciones a la hora de armar una computadora de escritorio, ya que se filtra los componentes compatibles, segun la placa madre elegida, también tiene una opción para la comparación entre componentes, y un listado de los componentes disponibles en la pagina.
